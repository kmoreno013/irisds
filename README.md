# **Iris Dataset Prediction App**
Welcome to the Iris Dataset Prediction App! This project demonstrates a machine learning model built to classify iris species (Setosa, Versicolor, Virginica) using the classic Iris dataset.

## **Features**
* Machine Learning Prediction: Classifies iris species based on input features (sepal and petal measurements).
* Interactive App: User-friendly interface to input measurements and get predictions in real-time.
